# 에코업 DCS-HM4AG-W 조사 메모

## 현재 판단

- 모델명과 제조사는 가격비교 페이지에서 확인된다.
- 습식분쇄와 미생물을 결합한 싱크대 내장형 제품으로 표시된다.
- 다나와 등록월은 2025년 8월이지만 실제 출시일과 동일하다고 단정하지 않는다.
- 공식 제품 페이지와 공식 설명서는 아직 확보하지 못했다.
- 기존 카탈로그에 연결된 세척 영상은 수동 재생과 채널 확인이 필요하다.

## 공개 데이터에서 바로 수정하지 않은 이유

현재 서버 카탈로그에는 주 1회 관리와 12분 소요가 들어 있지만, 이를 뒷받침하는
공식 주기와 시간 근거를 찾지 못했다. 새 조사 데이터에서는 두 값을 확정하지 않았다.
공식 설명서 또는 제조사 고객지원 답변을 확보한 뒤 공개 데이터를 수정한다.

## 추가로 필요한 실제 자료

사용자가 보유한 제품에서 다음 사진을 확보하면 식별과 자료 연결에 도움이 된다.

1. 본체 제조사 라벨 전체
2. 모델명과 인증번호가 선명한 확대 사진
3. 설명서 표지와 목차
4. 제품 또는 설명서의 QR·바코드
5. 미생물 제제 포장과 부품명

개인정보, 설치기사 연락처, 주소, 시리얼 번호는 공개 카탈로그에 저장하지 않는다.
