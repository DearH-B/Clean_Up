# 삼성 Bespoke AI 냉장고 정확 모델 3종

확인일: 2026-06-13

현재 상태: 데이터 검수 완료 · 출시 검증 중

공식 자료 기반 제품 식별과 관리 정보 검수는 완료했다. 다만 실제 Android 기기의
네트워크·오프라인 동작, 외부 링크 장애 대응, 접근성, 운영 모니터링과 법적 고지까지
통과하기 전에는 `출시 검증 완료`로 판단하지 않는다.

## 검수 결과

세 모델은 삼성전자 공식 제품 페이지에서 제품명, 모델 코드, 용량, 대표 이미지와 주요
기능을 확인했다. 삼성 공식 매뉴얼 API에서 세 모델 모두 동일한 사용자 매뉴얼을 제공하는
것도 확인했다.

- 문서: `RF9000F 2025 사용자 매뉴얼`
- 버전: 4.0
- 게시일: 2026-05-19
- 문서번호: `DA68-04836T-01`

따라서 세 제품은 단순한 시리즈 후보가 아니라 `공식 설명서 확인 모델`로 제공한다.
이 표시는 데이터 근거 수준이며 앱 전체의 출시 준비 완료를 의미하지 않는다.

## 공통 관리 범위

사용설명서에서 다음 항목을 직접 확인했다.

- 청소 전 전원 플러그 분리
- 문 재질별 외관 청소
- 내부, 박스와 선반의 물 청소 및 완전 건조
- 문 고무 패킹과 홈 청소
- 제품 뒷면 먼지를 1년에 한 번 정도 진공청소기로 제거
- 물 직접 분사, 락스·시너·알코올·벤젠과 거친 수세미 사용 금지
- 유리 선반에 따뜻한 물이나 충격을 가하지 않음

디스펜서, 오토필 물통, 먼지 필터, 위스키볼 아이스메이커와 카메라는 매뉴얼에서
`해당 모델만 참조`로 표시한다. 제품 페이지에서 해당 기능을 확인하지 못한 모델에는
임의로 노출하지 않는다.

## 앱 표시 원칙

- 제품 식별, 이미지와 스펙은 공식 제품 페이지 근거로 표시한다.
- 기본 관리법은 공식 사용자 매뉴얼 근거로 표시한다.
- 기능별 관리법은 실제 모델 적용이 확인된 경우에만 표시한다.
- 확인되지 않은 정수 필터, 탈취 필터와 부품번호를 범용 기본값으로 만들지 않는다.
