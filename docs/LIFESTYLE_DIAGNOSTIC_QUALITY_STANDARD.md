# 생활 문제 해결 정보 품질 기준

## 목적

생활 문제 해결 정보가 제품 종류, 재질, 안전 조건에 맞는지 추적하고 검수 상태를 사용자에게 정직하게 표시한다.

## 검수 상태

### 안전 편집 초안

- 일반적인 안전 원칙과 보수적인 관리 범위만 안내한다.
- 제품 분해, 전기 작업, 깊은 배관 작업을 안내하지 않는다.
- 재질과 제조사 안내를 먼저 확인하도록 표시한다.
- 공식 또는 공공기관 근거가 아직 연결되지 않은 상태다.

### 자료 검수 완료

- 제조사, 공공기관 또는 전문기관 자료를 한 개 이상 확보한다.
- 증상, 대처 순서, 금지 사항이 근거 자료와 충돌하지 않는지 검토한다.
- 적용 가능한 제품 종류와 재질을 명시한다.
- 검수자와 검수일을 기록한다.

### 공식 자료 확인

- 해당 제품 또는 제품군의 공식 설명서나 공식 고객지원 자료가 연결된다.
- 안내 단계별로 근거를 추적할 수 있다.
- 제품 고유 주의사항과 서비스센터 기준을 반영한다.
- 출처 URL과 최근 확인일을 유지한다.

## 필수 데이터

- 증상
- 상태 확인 질문
- 안전한 다음 행동
- 결과 유형
- 대처 순서
- 필요한 도구
- 금지 사항
- 즉시 중단 신호
- 추천 제품과 추천 이유
- 근거 유형
- 출처 제목과 URL
- 검수 상태와 검수일
- 적용 재질 또는 적용 전 확인사항

## 제품 추천 기준

- 정확한 상품명과 제조사를 기록한다.
- 사용 가능한 재질과 용도를 확인한다.
- 약품 혼합 위험과 보호구 조건을 함께 표시한다.
- 판매 링크가 검색 결과라면 정확 상품 링크로 오해하지 않도록 표시한다.
- 광고 또는 제휴 관계가 있으면 사용자에게 명확히 알린다.
- 판매 중단이나 링크 변경을 정기적으로 확인한다.

## 안전 기준

- 감전, 화재, 가스, 누수, 배터리 팽창은 사용 중단을 우선한다.
- 락스, 산성 세정제, 배수관 세정제 등은 혼합 금지를 표시한다.
- 전기 제품에 액체를 직접 분사하도록 안내하지 않는다.
- 사용자가 허용 여부를 알 수 없는 분해 작업은 안내하지 않는다.
- 깊은 배관 막힘, 구조 균열, 반복 누수는 전문가 점검으로 연결한다.

## 운영 절차

1. 사용자 검색과 증상 선택 데이터를 기준으로 검수 우선순위를 정한다.
2. 근거 자료를 등록하고 적용 제품과 재질을 확인한다.
3. 안전 문구와 대처 순서를 별도로 검토한다.
4. 앱 테스트와 실제 기기 화면을 확인한다.
5. 검수 상태와 검수일을 갱신한다.
6. 오래되거나 접근할 수 없는 출처는 재검수 대상으로 전환한다.
