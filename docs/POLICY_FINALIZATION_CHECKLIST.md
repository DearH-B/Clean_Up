## 개인정보처리방침에 반드시 들어갈 내용

- [ ] 앱이 기기에 저장하는 데이터
  - 공간, 제품, 제품 종류, 브랜드, 모델 후보, 구매일, 설치일, 메모
  - 관리 기록, 소모품 정보, 최근 검색, 사용자 설정
- [ ] 서버로 전송될 수 있는 데이터
  - 제품 검색어
  - 제품 정보 요청 내용
  - 누락 제품 제보
  - 앱 오류 제보
  - 사용성 개선 의견
- [ ] 서버로 동기화하지 않는 데이터
  - 계정 기능 도입 전 사용자 제품 목록
  - 계정 기능 도입 전 사용자의 집 구성 전체
- [ ] 카메라와 사진 사용 목적
  - QR, 바코드, 제품 라벨 인식
  - 제품 검색과 등록 보조
  - 사진 자체를 사용자 데이터베이스에 저장하지 않는다는 현재 정책
- [ ] 사용자가 입력하지 말아야 할 정보
  - 이름, 주소, 전화번호, 상세 거주지, 가족 정보 같은 불필요한 개인정보
- [ ] 보관 기간
  - 로컬 데이터는 사용자가 삭제하거나 앱을 삭제할 때까지
  - 서버 제보 데이터는 운영 정책에 따라 보관하며 사용자가 삭제 요청할 수 있어야 함
- [ ] 삭제 방법
  - 앱 설정의 데이터 관리에서 로컬 데이터 삭제
  - 문의 이메일을 통한 서버 제보 데이터 삭제 요청
- [ ] 제3자 제공 여부
  - 출시 시점에 제3자 제공이 없다면 없다고 명시
  - 외부 분석, 광고 SDK, 제휴 링크가 추가되면 별도 명시
- [ ] 국외 이전 여부
  - 서버와 외부 도구가 국내인지 해외인지 확정 후 명시

## 이용약관에 반드시 들어갈 내용

- [ ] 서비스 목적
  - 사용자가 보유한 가전, 가구, 생활용품의 관리 정보와 기록을 확인하도록 돕는 서비스
- [ ] 사용자 확인 책임
  - 실제 작업 전 제조사의 최신 설명서, 안전 지침, 리콜 안내를 확인해야 함
- [ ] 금지 행위
  - 허위 정보 제출
  - 위험한 수리법 또는 분해법 제출
  - 타인의 개인정보 입력
  - 서비스 방해
- [ ] 서비스 변경 가능성
  - 제품 데이터, 추천 제품, 링크, 기능은 운영 상황에 따라 변경될 수 있음
- [ ] 면책 범위
  - 앱은 생활 관리 보조 도구이며 공식 수리, 진단, 안전 인증을 대체하지 않음

## 제품 정보 책임 범위에 반드시 들어갈 내용

- [ ] 공식 자료 우선 원칙
  - 제조사 설명서, 리콜, 안전 공지, 서비스센터 안내가 앱보다 우선
- [ ] 자료 등급 표시 원칙
  - 정확한 모델 공식 자료
  - 같은 시리즈 또는 유사 제품 참고 자료
  - 일반 제품군 관리 자료
- [ ] 문제 확인 기능의 한계
  - 고장 확정이나 수리 지시가 아님
  - 누수, 연기, 전기 냄새, 과열, 감전 위험, 분해 필요 상황은 즉시 사용 중단 후 전문가 문의
- [ ] 사용자 제보 처리 원칙
  - 제보는 검토 후 반영
  - 검증되지 않은 제보는 공식 자료처럼 표시하지 않음
