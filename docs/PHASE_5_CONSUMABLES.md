# Phase 5: 소모품과 필터 관리

상태: 핵심 구현 완료

완료일: 2026-06-11

## 목표

제품 청소 기록과 필터·소모품 교체 기록을 분리한다.

사용자는 제품 상세에서 마지막 교체일과 다음 확인 시기를 바로 볼 수 있어야 한다.

## 구현 내용

- 필터, 세척제, 보충품, 교체 부품, 기타 유형
- 소모품 이름과 교체 주기
- 호환 정보
- 부품 번호
- 마지막 교체일과 다음 교체일
- 사용자 메모
- 구매 링크
- 광고·제휴 여부
- 소모품 직접 추가, 수정과 삭제
- `교체했어요` 또는 `보충했어요` 빠른 기록

소모품을 교체하면 제품 데이터의 해당 소모품 일정과 전체 관리 기록이 함께
갱신된다.

## 제품군 기본 후보

다음 제품군은 모델을 정확히 모르는 경우에도 확인할 만한 소모품 후보를 제안한다.

- 냉장고
- 공기청정기
- 정수기
- 에어컨
- 식기세척기
- 세탁기
- 건조기

기본 후보는 정확한 호환 제품을 단정하지 않는다.

- `모델별 전용 필터 확인 필요`
- `해당 기능이 있는 모델만 해당`
- `제조사가 허용한 제품 확인`

같은 표현으로 정보 수준을 명시한다.

## 기록 단순화

일반 청소 기록의 주된 목적은 마지막 청소일을 기억하는 것이다.

- 제품 상단에서 마지막 청소일 표시
- `오늘 청소` 버튼 한 번으로 기록
- 상세 기록 화면은 날짜와 메모만 먼저 표시
- 유형, 시간, 사용 용품, 결과와 다음 확인일은 선택 영역에 배치
- 전체 기록 화면의 총 관리 시간 통계 제거

필터 교체와 소모품 교체는 제품의 마지막 청소일을 변경하지 않는다.

## 안전과 신뢰 기준

- 부품 번호가 확인되지 않으면 임의 번호를 제공하지 않는다.
- 모델 호환이 확인되지 않으면 전용 제품처럼 표시하지 않는다.
- 일반 먼지 필터처럼 세척 대상일 수 있는 부품을 무조건 교체 대상으로 안내하지 않는다.
- 구매 링크가 광고 또는 제휴 링크면 명확히 표시한다.
- 제조사 설명서와 제품 라벨의 호환 정보가 앱 안내보다 우선한다.

## 후속 데이터 작업

기능 구조는 완성됐지만 실제 서비스 품질을 위해 다음 데이터 수집이 필요하다.

1. 모델별 공식 필터와 부품 번호
2. 제조사 권장 교체 주기
3. 공식 구매 페이지
4. 단종 및 대체 부품 정보
5. 공식 호환 여부의 출처와 확인일

이 데이터는 Phase 2 카탈로그 품질 작업과 계속 병행한다.
