# 제품 데이터 품질 기준

이 문서는 앱에 등록하는 가전, 가구, 소모품 자료의 공통 수집 및 검수 기준이다.
모든 제품에 동일한 양의 정보를 요구하지 않으며, 제품의 복잡도와 위험도에 맞는 기준을 적용한다.
확인되지 않은 내용은 추측해서 채우지 않는다.

## 적용 원칙

- 모든 제품에는 `공통 최소 기준`만 필수로 적용한다.
- 전기, 물, 열, 화학물질처럼 위험 요소가 있는 제품은 강화 기준을 적용한다.
- 제품에 해당하지 않는 항목은 누락이 아니라 `해당 없음`으로 처리한다.
- 작은 소모품에 보증기간, 설치일, 오류 코드 같은 불필요한 정보를 요구하지 않는다.
- 정보가 풍부한 정도와 정보가 정확한 정도를 별도로 평가한다.

## 1. 기본 식별 정보

- 제품군: 가전 / 가구 / 소모품
- 카테고리와 세부 유형
- 제조사와 브랜드
- 제품명과 모델명
- 출시 연도 또는 판매 시기
- 제품을 식별할 수 있는 대표 이미지

## 2. 식별 정확도

사용자가 모델명을 모르더라도 등록할 수 있도록 다음 단계로 구분한다.

- 정확한 모델: 모델명이 공식 자료와 일치함
- 유사 제품군: 같은 제조사의 구조와 관리법이 유사한 제품군
- 일반 제품: 제조사나 모델을 특정하지 않은 일반 정보
- 미확인 제품: 식별에 필요한 정보가 부족함

유사 제품 자료를 사용할 때는 정확한 모델 전용 정보처럼 표시하지 않는다.

## 3. 사용 환경

- 주로 사용하는 구역
- 재질, 크기, 설치 방식
- 전원, 급수, 배수 등 필요한 조건
- 어린이와 반려동물 관련 주의사항

해당되지 않는 항목은 비워두거나 적용 대상이 아님을 표시한다.

## 4. 관리 방법

- 평상시 관리
- 정기 관리
- 오염 또는 상태별 관리
- 단계별 작업 순서
- 예상 소요 시간
- 분해 또는 전문 작업 필요 여부

관리 방법은 일반 제품군 정보와 해당 모델 전용 정보를 구분한다.

## 5. 관리 주기

- 제조사 권장 주기
- 사용량 또는 상태에 따른 조정 조건
- 사용자가 직접 변경 가능한 권장 주기
- 마지막 관리일과 다음 권장일 계산 가능 여부

공식 자료에 주기가 없으면 임의의 숫자를 공식 권장 주기로 표시하지 않는다.

## 6. 안전 및 주의사항

- 사용하면 안 되는 세제와 도구
- 물, 열, 전기 관련 위험
- 변색, 부식, 변형, 표면 손상 가능성
- 전원 차단이나 보호 장비가 필요한 작업
- 사용자가 직접 하면 안 되는 작업
- 전문가 또는 공식 서비스가 필요한 작업

안전 정보는 편의 정보보다 우선한다.

## 7. 연관 소모품과 관리 도구

- 필요한 관리용품
- 호환 소모품
- 규격 또는 부품번호
- 교체 방법과 권장 주기
- 정품과 호환품 구분
- 판매 종료 또는 단종 여부
- 구매 링크의 광고 및 제휴 여부

특정 상품을 추천할 때는 추천 근거와 광고 또는 제휴 관계를 명확히 표시한다.

## 8. 수명과 상태 정보

- 구매일과 설치일
- 보증기간
- 현재 사용 상태
- 주요 이상 증상
- 수리와 부품 교체 이력
- 제조사가 안내한 교체 또는 점검 조건

근거 없는 예상 수명은 확정 정보로 제공하지 않는다.

## 9. 출처 기준

자료는 다음 순서로 우선 사용한다.

1. 해당 모델의 공식 제품 페이지와 공식 설명서
2. 같은 제조사의 공식 고객지원 및 동일 제품군 자료
3. 정부, 공공기관, 인증기관의 안전 자료
4. 다른 제조사의 유사 제품 공식 자료
5. 신뢰할 수 있는 전문 자료
6. 사용자 제보와 후기

각 자료에는 다음 정보를 저장한다.

- 출처 이름
- 원문 링크
- 자료 종류
- 적용 대상
- 확인한 날짜
- 자료가 수정되거나 사라졌는지 확인할 상태

블로그, 쇼핑몰 설명, 영상, 후기는 공식 자료를 보완할 수 있지만 단독 공식 근거로 사용하지 않는다.

## 10. 정보 신뢰도

- A: 해당 모델의 공식 자료
- B: 같은 제조사의 동일 또는 유사 제품군 공식 자료
- C: 공공기관 자료 또는 다른 제조사의 유사 제품 공식 자료
- D: 일반적인 전문 관리 정보
- 미검증: 출처가 충분하지 않은 사용자 정보

하나의 제품 안에서도 관리법, 주기, 소모품 정보마다 신뢰도 등급이 다를 수 있다.

## 제품군별 추가 항목

### 가전

- 전원과 소비전력
- 필터, 배수, 급수 구조
- 펌웨어와 오류 코드
- 공식 부품과 서비스 정보

### 가구

- 소재와 표면 마감
- 허용 하중
- 권장 온도와 습도
- 변색과 뒤틀림 위험
- 조립, 보수, 부분 교체 방법

### 소모품

- 규격과 호환 제품
- 제조일, 개봉일, 사용기한
- 보관 조건
- 잔량과 교체 조건
- 폐기 및 분리배출 방법

## 데이터 완료 판정

### 공통 최소 기준

가구, 가전, 소모품의 크기와 관계없이 다음 항목만 기본적으로 요구한다.

- 사용자가 구분할 수 있는 이름과 제품 유형
- 주요 용도 또는 사용 방법
- 꼭 알아야 할 주의사항 또는 `특별한 주의사항 없음`
- 교체, 보관, 폐기 정보 중 해당되는 내용
- 정보의 출처 또는 사용자 직접 등록 표시
- 식별 정확도와 최종 확인일

### 강화 기준 적용 대상

다음 조건 중 하나라도 해당하면 관련 세부 정보를 추가로 검수한다.

- 전기, 배터리, 가스, 급수 또는 배수 기능이 있음
- 화학 성분이나 인체 접촉 관련 주의가 필요함
- 잘못 관리하면 화재, 감전, 누수, 제품 손상이 발생할 수 있음
- 전용 필터나 특정 규격의 교체 부품이 필요함
- 제조사나 모델에 따라 관리법이 크게 달라짐
- 설치, 분해 또는 전문 서비스가 필요함

강화 기준 제품은 대표 이미지, 공식 출처, 관리 방법, 주기, 안전사항,
호환 소모품 정보를 제품 특성에 맞게 추가한다.

### 소형 소모품 간소화 기준

수세미, 청소포, 브러시, 쓰레기봉투처럼 구조가 단순한 소모품은 다음 정보만으로
`기본 검수 완료`가 가능하다.

- 제품명 또는 일반 명칭
- 용도와 적합한 사용 대상
- 재질, 크기, 용량 또는 규격 중 필요한 항목
- 사용 시 주의사항
- 교체, 세척, 보관 또는 폐기 방법 중 해당되는 항목
- 제조사 표기, 포장 정보, 공식 판매 정보 또는 사용자 직접 등록 표시

단순 소모품에는 모델명, 출시 연도, 보증기간, 설치 정보, 수리 이력을 요구하지 않는다.

### 검수 상태

- 기본 검수 완료: 공통 최소 기준을 충족함
- 상세 검수 완료: 제품 특성에 필요한 강화 기준까지 충족함
- 작성 중: 필수 정보 일부가 부족함
- 검수 필요: 출처 또는 작성 내용의 재확인이 필요함
- 사용자 등록: 개인이 직접 입력했으며 운영 검수를 거치지 않음

공개 카탈로그에는 최소한 `기본 검수 완료` 상태의 데이터만 반영한다.

## 자료 수집 절차

1. 제품군과 대표 제품을 선정한다.
2. 공식 제품 페이지와 설명서를 확보한다.
3. 식별 정보, 관리법, 주기, 주의사항, 소모품을 분리해서 기록한다.
4. 일반 정보와 모델 전용 정보를 구분한다.
5. 각 정보에 출처, 확인일, 신뢰도를 연결한다.
6. 완료 판정 체크리스트로 1차 검수한다.
7. 다른 사람이 출처와 작성 내용을 대조하여 2차 검수한다.
8. 검수가 끝난 데이터만 앱의 공개 카탈로그에 반영한다.

## 운영 원칙

- 빈칸보다 잘못된 정보가 더 위험하다.
- 자료가 없으면 `확인되지 않음`이라고 표시한다.
- 유사 제품 정보는 참고 정보임을 사용자가 알 수 있게 한다.
- 공식 자료가 변경되거나 링크가 사라질 수 있으므로 정기적으로 재검수한다.
- 안전과 제품 손상에 관련된 내용은 반드시 출처를 다시 확인한다.
- 제품 판매 링크와 정보 제공 영역을 분리하여 신뢰성을 지킨다.
