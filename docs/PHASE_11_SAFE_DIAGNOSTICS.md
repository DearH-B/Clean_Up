# Phase 11: 안전한 문제 확인

## 방향

이전 로드맵의 커뮤니티 기능은 제외했다. 제품 관리 앱의 핵심 가치와 직접 연결되는
증상 확인 기능만 구현한다. 앱은 고장을 확정하거나 수리를 지시하지 않는다.

## 구현 내용

- 제품군별 대표 증상 선택
- 안전 관련 확인 질문을 일반 관리보다 먼저 표시
- 직접 확인, 설명서 확인, 소모품 확인, 사용 중단, 전문가 문의로 결과 구분
- 누수, 연기, 전기 냄새와 분해가 필요한 상황은 사용 중단 우선
- 결과에서 제품별 공식 설명서, 고객지원과 서비스센터 연결
- 냉장고, 세탁기, 식기세척기, 음식물처리기 전용 흐름
- 그 외 제품을 위한 보수적인 범용 흐름

## 안전 원칙

- 공식 설명서와 제조사 안내가 항상 우선한다.
- 출처가 없는 수리 절차를 생성하지 않는다.
- 전기 부품, 펌프, 냉매 배관과 본체 분해를 권하지 않는다.
- 사용자가 답을 모르면 위험하지 않은 쪽으로 단정하지 않는다.

## 출시 전 실제 검증

- 각 제품군 5개 이상 실제 증상 문구 검토
- 서비스센터와 공식 설명서 링크 유효성 확인
- 안전 담당자 또는 제품 수리 전문가의 문구 검토
- TalkBack으로 질문과 결과 읽기 순서 확인
