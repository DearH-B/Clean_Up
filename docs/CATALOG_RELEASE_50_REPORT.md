대표 이미지를 수집하고 공식 설명서 API에서 사용자 매뉴얼 PDF를 연결했다.

## 에뮬레이터 검증

Android 에뮬레이터에서 신규 인덕션 `CC80H63G1HS`를 모델명으로 검색하고 다음
흐름을 확인했다.

1. 검색 결과에 `Bespoke AI 인덕션 (플렉스존)`과 검수일 표시
2. 확인 화면에 공식 설명서 확인 모델, 2026년, 플렉스존, 공식 제품 페이지 표시
3. 주방 공간에 정확 모델 등록 성공
4. 상세 화면에 공식 자료 상태, 인덕션 전용 준비물과 관리 순서 표시

앱 실행, 검색, 확인, 저장, 상세 화면 이동 과정에서 검은 화면이나 예외가 발생하지
않았다.

## 연식 기준

공식 페이지에는 제품 출시일이 별도 필드로 제공되지 않는 경우가 있다. 이때 삼성
모델 코드의 연식 문자로 출시 연식을 기록하고, 공식 페이지 이미지의 연도는
`pageAssetYear`로 분리했다. 출시 전 운영 검수에서 제조사 발표 자료와 한 번 더
대조한다.
