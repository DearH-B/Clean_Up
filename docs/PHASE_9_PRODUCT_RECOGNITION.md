# Phase 9: 제품 인식과 검색 보강

## 완료 범위

- 제품 등록 화면에서 카메라 또는 사진으로 제품 라벨 촬영
- ML Kit OCR로 라벨 문자 인식
- 모델명, 형명, 품번 주변 문자를 우선 후보로 제시
- 선택한 후보로 앱 카탈로그 즉시 검색
- 모델명 한두 글자 오타에 대한 유사 검색
- 사진은 앱에 별도로 저장하지 않음

## 사용자 흐름

1. 내 제품에서 공간을 선택하고 제품 추가를 누른다.
2. `라벨 촬영`을 선택한다.
3. 제품 옆면 또는 뒷면의 모델명 라벨을 촬영한다.
4. 인식된 모델명 후보 중 라벨과 같은 문자를 고른다.
5. 검색 결과를 확인하고 제품을 등록한다.

## 한계와 다음 검증

- 훼손된 라벨, 반사, 작은 글자는 인식률이 낮을 수 있다.
- 현재는 라틴 문자와 숫자로 구성된 국내 가전 모델명에 최적화했다.
- 실제 기기에서 대표 제품 라벨 20종을 촬영해 성공률을 기록해야 한다.
- OCR 결과가 없을 때는 기존 검색과 직접 입력을 그대로 사용할 수 있다.
