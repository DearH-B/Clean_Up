# Phase 2: 제품 카탈로그 품질 강화

상태: 진행 중

완료 기준은 [실제 출시 품질 기준](RELEASE_QUALITY_STANDARD.md)을 따른다.
카탈로그의 `verified`는 공식 자료 기반 데이터 검수 완료를 뜻하며, 실제 기기와 운영
환경까지 포함한 출시 검증 완료를 뜻하지 않는다.

Phase 2는 제품 수를 무작정 늘리는 단계가 아니다. 제품 정보가 어디에서 왔고,
어느 수준까지 검수됐는지 추적할 수 있는 기반을 먼저 만든다.

## 완료된 기반 작업

### 제품 데이터 규칙

카탈로그 제품은 다음 정보를 관리한다.

- 내부 제품 ID
- 제품 종류, 표시 이름, 브랜드, 제조사와 모델명
- 처리방식과 주요 스펙
- 관리법 수준과 검수 상태
- 마지막 출처 확인일
- 설명서, 고객지원, 출시 연도, 단종 여부 등의 선택 정보
- 설치 방식과 필터 및 소모품

기존 단일 `sourceTitle`, `sourceUrl` 필드는 이전 앱 데이터와의 호환을 위해
유지한다. 새 카탈로그 데이터는 `sources`를 기준으로 작성한다.

### 다중 출처

제품 하나에 여러 출처를 연결할 수 있다.

각 출처에는 다음 내용을 저장한다.

- 출처 ID
- 제목과 URL
- 출처 유형
- 게시 주체
- 확인일
- 근거로 사용한 정보
- 공식 여부
- 현재 유효 여부

지원 출처 유형:

- 공식 제품 페이지
- 공식 설명서
- 공식 영상
- 공식 고객지원
- 판매 페이지
- 가격비교 스펙
- 설치업체 자료
- 유사 제품 참고
- 일반 제품군 관리법

### 근거 추적

- `specSourceIds`: 스펙 항목과 출처 연결
- `stepSourceIds`: 관리 단계와 출처 연결
- 숫자가 포함된 스펙은 반드시 출처가 연결되어야 한다.
- 존재하지 않는 출처 ID를 참조하면 서버가 카탈로그 로드를 거부한다.

### 검수 이력

`reviewHistory`에 상태, 검수자, 검수일과 메모를 기록한다.

- `draft`: 조사 중이며 공개하지 않음
- `reviewed`: 모델과 안전 범위를 1차 검수함
- `verified`: 공식 자료와 주요 정보를 교차 확인함

공개 상태와 마지막 검수 이력의 상태가 다르면 카탈로그 로드를 거부한다.

### 재검수 판단

다음 출처는 재검수 대상으로 판단할 수 있다.

- `isActive`가 `false`인 출처
- 마지막 확인 후 정해진 기간이 지난 출처

현재 기본 재검수 기준은 180일이다.

## 현재 마이그레이션 상태

- 에코업 DCS-HM4AG-W의 제품 스펙과 공식 영상을 별도 출처로 분리
- 삼성 Bespoke AI 냉장고 정확 모델 3종을 공식 제품 페이지와
  `RF9000F 2025` 사용자 매뉴얼로 검수
- RM80F91H1W의 UV 청정탈취 필터 권장 사용기간 10년을 모델 전용
  소모품으로 연결하고 확인되지 않은 부품번호는 제외
- 일반 냉장고 관리법을 비공식 제품군 가이드로 명시
- 일반 공기청정기 관리법을 비공식 제품군 가이드로 명시
- 사용자 제품 저장 시 다중 출처 정보 유지
- 제품 상세 화면에서 출처별 공식 여부, 유형, 게시 주체, 확인일과 근거 표시

## 제품군별 관리법

모델을 찾지 못했거나 추천 제품에서 바로 등록한 경우에도 모든 제품에 같은 범용
청소법을 사용하지 않는다.

현재 제품군별 템플릿을 제공하는 항목:

- 냉장고
- 식기세척기
- 음식물처리기
- 전자레인지
- 세탁기
- 건조기
- 공기청정기
- 에어컨
- 소파
- 매트리스
- 싱크대

각 템플릿은 관리 주기, 준비물, 관리 지점, 금지 행동과 단계를 별도로 가진다.
예를 들어 냉장고는 선반과 문 고무패킹을, 식기세척기는 필터와 배수구 주변 및 분사
노즐을 중심으로 안내한다.

템플릿이 준비되지 않은 제품은 임의의 중성세제와 공통 순서를 보여주지 않는다.
대신 세부 관리법이 준비 중임을 표시하고 제조사 설명서를 우선하도록 안내한다.

이전 버전에서 공통 관리법으로 등록한 제품은 앱을 불러올 때 해당 제품군 템플릿이
있으면 사용자 별칭과 구매 정보 등을 유지한 채 자동 갱신한다.

## 다음 작업

Phase 2의 다음 작업은 카탈로그 확장이다.

1. 제품군별 조사 템플릿 확정
2. 음식물처리기, 냉장고, 세탁기, 에어컨, 공기청정기 순서로 확장
3. 제품군별 일반 관리법 최소 1개 확보
4. 대표 브랜드와 모델 조사
5. 최소 25개의 검수된 카탈로그 항목 확보

2026-06-13 기준 공개 카탈로그는 17개이며, 이 중 공식 자료 기반 데이터 검수를 마친
`verified` 정확 모델은 삼성 냉장고 3개다. 세 모델은 아직 출시 검증 중이며,
Phase 2 수량 목표까지 8개가 남았다.

제품을 추가할 때는 화면보다 출처와 검수 품질을 먼저 확인한다.
